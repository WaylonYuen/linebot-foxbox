# linebot-sample `Line 聊天機器人 python 範例`

###### 【 About 關於 】
此專案為 Line 聊天機器人`(以下簡稱為Linebot)`Python 範例, 令使用 LINE Messaging API for Python 開發機器人變得容易, 並且您可以在幾分鐘內創建一個範例機器人.
